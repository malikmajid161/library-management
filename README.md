# 📚 LibMaster

**LibMaster** is a premium, high-performance Library Management System built with **Flutter**. It offers a seamless experience for managing books, members, and authors with a stunning, modern UI and robust offline-first storage using **Hive**.

### 🔗 Live Demo
Experience the app live at:  
**[https://malikmajid161.github.io/library-management/](https://malikmajid161.github.io/library-management/)**

---

## ✨ Features

- **🚀 Performance-Driven**: Lightning-fast performance with Hive NoSQL database.
- **📱 Responsive Design**: Beautifully optimized for Web, Android, iOS, and Desktop.
- **📂 Full CRUD Operations**: Manage Books, Categories, Publishers, Authors, and Locations.
- **🔐 Multi-User Support**: Individualized storage per user for data privacy.
- **🎨 Premium Aesthetics**: Modern design system with smooth transitions and glassmorphism.
- **🔍 Advanced Search**: Real-time filtering and searching through your entire library.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev)
- **Database**: [Hive](https://pub.dev/packages/hive) (Local NoSQL)
- **State Management**: Provider
- **Theming**: Custom Premium Design System
- **CI/CD**: GitHub Actions

## 📦 Getting Started

### Prerequisites
- Flutter SDK (Latest stable version)
- Dart SDK

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/malikmajid161/library-management.git
   ```
2. Navigate to the project directory:
   ```bash
   cd library-management
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run -d chrome
   ```
